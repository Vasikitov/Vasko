Dodgeball Masters  _GAME_

Description

Dodgeball Masters is a frenetic game where two teams will compete
until one stands above the other by process of elimination.
There are 8 different characters, where each one of them has its own strengths and weaknessess.
It's a 3v3 gameplay, where the process of elimination is the only way
to achieve victory.

This game was a group project and was made completely from scratch, using only
C++ and OpenGL for the creation and integration of our own graphics library.

Released: 2020
