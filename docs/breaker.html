Breaker

Description

This is one of the classics of all time. I had it as University project and had to make a similar version of Block Breaker with a bit of touch. Added double pads and different tile positioning each time you play the game, so we can keep it fresh and varied.

The game is fully programmed from scratch using C++ and SFML library.

Released: 2018
